# Rnnoise window demo

Get Wsl on window

```bash
cd /mnt/wsl
```

```bash
git clone https://github.com/xiph/rnnoise.git
```

```bash
sudo apt-get install libtool
sudo apt install autoconf
```
```bash
./autogen.sh
./configure
make
```

```bash
explorer.exe .
```

```bash
./examples/rnnoise_demo <noisy speech> <output denoised>
```
